主题

rollup-plugin-add-banner为打包文件添加 Banner

一个 Rollup 插件，用于在输出文件顶部添加 banner 注释

快速上手
GitHub 仓库
🔧

简单集成

与 Rollup 2.x、3.x 和 4.x 无缝配合，零配置即可使用

📦

许可证就绪

轻松为打包文件添加版权声明和许可证信息

🎯

灵活内容

支持单行和多行 banner 内容

⚡

TypeScript 优先

完整的 TypeScript 支持和类型定义

🗺️

Source Map 支持

添加 banner 时自动调整 source map

🪶

轻量级

无运行时依赖，体积最小

快速开始 ​

安装 ​

bash
# pnpm
pnpm add -D rollup-plugin-add-banner

# npm
npm install -D rollup-plugin-add-banner

使用 ​

js
import addBanner from 'rollup-plugin-add-banner'

export default {
  input: 'src/index.js',
  output: {
    file: 'dist/index.js',
    format: 'es'
  },
  plugins: [
    addBanner({
      content: '/*! ${name} v${version} (c) ${author} */'
    })
  ]
}

为什么选择 rollup-plugin-add-banner？ ​

  • 模板变量：自动从 package.json 解析 ${name}、${version}、${author}、${license}
  • 简单 API：只需提供 banner 内容即可
  • 全版本兼容：支持 Rollup 2.x、3.x 和 4.x
  • 多 Banner 支持：不同文件类型使用不同的 banner
  • Include/Exclude：使用 glob 模式精确控制
  • 100% 测试覆盖：经过充分测试，可靠稳定
